# CRAFT
This repository contains the source code for the CRAFT framework.

The source code provided in this repository enables the model to achieve extremely high inference speed (268.70 FPS) on the VisA, MVTec, and RealIAD datasets for image-level anomaly detection. Furthermore, its performance is comparable to or even better than heavyweight base models such as WideResNet-50, with an average pixel-level F1 score of 0.923.
